#LOAD TESTING ON VARIOUS BUSINESS PAGES OF INSTAGRAM#

1. Clone Repo
2. Open in Pycharm
3. Setup python env and add pytest, pytest-html, selenium, pytest-xdist libraries
4. Run Command  pytest .\testCases\  --browser chrome


Screenshots -> All Screenshots captured during excution
Logs -> Logs for each step executions
Configurations -> Basic common variables
pageObjects -> Individual Page actions
testCases -> Test Flows
utilities -> Logger and readProperties to read config file
